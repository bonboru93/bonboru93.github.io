kpatch使用方法

2019-11-25
  • 编译kpatch
cd /lib/modules/$(uname -r)
ln -s kernel_build ./build
ln -s kernel_build ./source
git clone https://github.com/dynup/kpatch.git
cd kpatch-master
make
cd kmod/core
make
  • 创建目录
    1. 编译目录
    git clone gbcc_url
mv gbcc kernel_build
git checkout working
sh build.sh > /dev/null
    1. 未编译目录
    git clone gbcc_url
mv gbcc kernel_origin
git checkout working
    1. patch目录
    git clone gbcc_url
mv gbcc kernel_kpatch
git checkout kpatch
  • 使用脚本
diff -x ".git" -Nuar kernel_origin kernel_kpatch > bbu.patch
cat bbu.patch
CACHEDIR=`pwd`/cache kpatch-master/kpatch-build/kpatch-build -t vmlinux bbu.patch -v kernel_build/vmlinux -s kernel_build
rm -rf cache

脚本会判断内核是否支持livepatch，如果支持则产出的模块以livepatch打头，不支持则以kpatch打头。
